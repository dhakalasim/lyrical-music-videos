# lyrical-music-videos

A responsive browser app for building lyrical music video previews from uploaded audio and images.

## Features

- Upload audio tracks and image slides.
- Edit captions and slide durations.
- Preview image slideshow synced with audio playback.
- Download a project file with slide settings.

## Run locally

1. Open `index.html` in your browser.
2. Upload an audio file and one or more images.
3. Edit captions, then preview.

> Note: This version provides a responsive editor and preview experience in the browser.
